# Microsoft Azure 虛擬機器食譜

本書以範例形式指引如何在 [Microsoft Azure](https://azure.microsoft.com/) 上完成各種虛擬機器服務的任務，並且藉此瞭解 Microsoft Azure 的各種服務使用。

目錄：
* 環境、操作工具準備
* 建立及使用虛擬機器
* 虛擬機器的系統管理
* 高穩定架構以及負載平衡
* 虛擬網路架構
* 維運及監控
* 架構參考
